Experiment Priority Matrix
Impact vs Effort scoring for all planned tests
01
Fix the Search → Book conversion gap first
The 60.7% drop from search to transaction is the single largest revenue opportunity. A 10pp improvement at this stage = ~560 additional converted users per cohort = significant GMV impact.
02
Build a D0–D7 lifecycle engagement engine
Invest in a CRM-driven push/SMS sequence calibrated to user type. Power riders get upsell messages. Browsers get social proof. One-timers get win-back offers. Move from broadcast to behavioral.
03
Ship "Rapido Pass" subscription for commuters
Users with 3+ W1 sessions are commuters — highest LTV segment. A ₹199/month pass creates predictable revenue, improves supply planning, and locks in habit before competitors.
04
Track engagement metrics, not just bookings
Add "search-to-book ratio" and "D3 session rate" as leading indicators of W2 retention. Current dataset only captures binary events — granular funnel timing would improve experiment velocity.
